## 🔧 解决外网访问问题

### 方法 1: 配置云服务器安全组（推荐）

如果你使用的是云服务器，需要：

#### 阿里云
1. 登录 [ECS控制台](https://ecs.console.aliyun.com/)
2. 选择你的实例
3. 点击"安全组" → "配置规则"
4. 添加入方向规则：
   - **端口范围**: 3000/3000
   - **授权对象**: 0.0.0.0/0
   - **协议类型**: TCP
   - **优先级**: 1

#### 腾讯云
1. 登录 [CVM控制台](https://console.cloud.tencent.com/cvm)
2. 选择你的实例
3. 点击"安全组" → "修改规则"
4. 添加入站规则：
   - **端口**: 3000
   - **来源**: 0.0.0.0/0
   - **协议**: TCP

#### 其他云服务商
参考各自的文档，开放 3000 端口的入站流量。

### 方法 2: 配置系统防火墙

#### 使用 ufw (Ubuntu/Debian)

```bash
# 开放 3000 端口
sudo ufw allow 3000/tcp

# 查看状态
sudo ufw status
```

#### 使用 firewall-cmd (CentOS/RHEL)

```bash
# 开放 3000 端口
sudo firewall-cmd --permanent --add-port=3000/tcp

# 重载防火墙
sudo firewall-cmd --reload

# 查看开放的端口
sudo firewall-cmd --list-ports
```
